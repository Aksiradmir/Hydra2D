# Hydra2D
==============================================================

Implementation of the 2D SPH approach in the Fluid Dynamics simulations. 


Dependencies
------------

- OpenMP (GCC already contains it)

Installation
------------

On Ubuntu:
```bash
git clone https://github.com/Aksiradmir/Hydra2D.git
cd Hydra2D
mkdir build
cd build
cmake ..
make
./Hydra2D
```
